# Structure

How a piece is built — openings that earn attention, transitions that carry weight, evidence that lands. Use when outlining, drafting long-form, or fixing a piece that wanders.

## Entries (by priority)

- [[lead-with-the-news]] (p1) — Put the most important sentence first. Reader gets the point before deciding whether to keep reading. Background, caveats, and reasoning come after.
